# 自動化營運手冊 — 排程、輸出規則、下單流程

> 建立日期：2026-07-03
> 原則：**系統只發訊號與清單，永遠不自動下單。** 最後一步（券商委託）由你執行。

## 1. 排程總覽（Windows 工作排程器，已註冊）

| 任務 | 時間 | 內容 |
|---|---|---|
| `TWQuant-Intraday` | 週一~五 08:55 → 13:35 自動結束 | 盤中監控：持股報價輪詢（5 分鐘一次）、事件警示、心跳 |
| `TWQuant-EOD` | 週一~五 17:40 | 更新全市場資料 → 波段掃描（紙上）→ 月底加發動能換股清單 |

- 兩個任務都設了 WakeToRun（自動喚醒電腦）與 StartWhenAvailable（錯過補跑）。
- **電腦必須開機（可休眠）且能上網**；關機就不會跑。
- 全部輸出送 Discord（環境變數 `DISCORD_WEBHOOK_URL`）；本機日誌在 `logs/`。

## 2. Discord 輸出規則（事件驅動，控制雜訊）

| 時間 | 訊息 | 規則 |
|---|---|---|
| 08:55 | :sunrise: 盤前摘要 | 持股清單＋今日待辦；換股日附完整買賣清單與「漲停放棄」規則 |
| 09:05 | :bell: 開盤回報 | **僅換股日**：逐筆回報開盤價，開盤漲幅 ≥ 9.5% 的買單標記「放棄」 |
| 盤中 | :warning: 持股警示 | 持股單日 ≤ −8%（每檔每日最多一次）；**資訊性質，月調倉紀律不因此動作** |
| 盤中 | :chart_with_downwards_trend: 大盤警示 | TAIEX 單日 ≤ −3%（每日一次） |
| 10:30 / 11:30 / 12:30 | :hourglass: 心跳 | 一行：持股數＋估當日損益。**心跳沒出現 = 系統掛了**，去看 logs/intraday.log |
| 13:35 | :checkered_flag: 收盤摘要 | 每檔持股當日漲跌與估損益 |
| 17:40+ | :gear: 盤後報告 | 資料更新結果 → 波段掃描（標註紙上觀察）→ 月底加發換股清單 |

注意：盤中報價來自 Yahoo，可能延遲 5~15 分鐘；警示門檻寫在 `scripts/monitor_intraday.py` 開頭常數。

## 3. 下單流程（唯一需要你動手的部分）

### 月度換股（動能組合，每月一次）
```
月底最後交易日 17:40   Discord 收到「動能組合月度換股清單」
        ↓
當晚                  你核對清單（處置股、異常訊息自行剔除）
        ↓
次一交易日 08:30-09:00  券商 App 下單：
                        ① 賣單先掛（開盤市價賣出全部「SELL」清單）
                        ② 買單後掛（開盤市價或漲跌停範圍內限價）
        ↓
09:05                 Discord 開盤回報：漲幅 ≥ 9.5% 的買單 → 刪單放棄（缺額留現金）
        ↓
成交後                更新 holdings.csv（symbol,shares，以券商回報為準）
                      若權益變動大，順手更新 settings.json 的 equity
        ↓
之後整月              不動作。盤中警示只是資訊，紀律是月調倉才動。
```

### 每日（不需動作）
- 08:55~13:35 監控訊息看看就好。
- 17:40 盤後波段掃描 = **紙上觀察**（該策略 KPI Gate 未過，見 STRATEGY.md 第 13 節）。

## 4. 檔案與設定

| 檔案 | 用途 |
|---|---|
| `holdings.csv` | 你的實際持股（symbol,shares）——成交後手動維護，系統靠它算損益 |
| `settings.json` | `equity`：目前帳戶權益，用於計算建議股數 |
| `logs/intraday.log`、`logs/eod.log` | 排程執行日誌（除錯用） |
| 環境變數 `DISCORD_WEBHOOK_URL`、`FINMIND_TOKEN` | 已設定（User 層級） |

## 5. 風險與紀律（再說一次）

1. 動能組合回測 MDD −50%，且資料有存活者偏差（回測偏樂觀）。**建議先紙上跑 1~3 個月**，實際投入時控制在可承受腰斬的資金比例。
2. 盤中警示不是出場訊號。看到 −8% 警示就手動砍倉 = 破壞已驗證的紀律，把系統變成心情交易。
3. 排程改時間／停用：工作排程器 GUI（taskschd.msc）找 `TWQuant-*`，或
   `Disable-ScheduledTask -TaskName 'TWQuant-Intraday'`。
